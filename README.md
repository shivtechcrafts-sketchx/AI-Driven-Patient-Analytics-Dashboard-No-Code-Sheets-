## ‚ö° What is MedFlow AI?

MedFlow AI is an intelligent hospital analytics system that automatically converts raw patient admission data into live dashboards using AI Agents inside n8n.

It behaves like a virtual hospital data analyst that:

- Reads patient records  
- Generates KPIs automatically  
- Detects disease trends  
- Calculates billing insights  
- Updates Sheets dynamically  
- Responds to chat queries
